Xác thực
  • JWT HS256, TTL theo JwtService
  • Cookie: jwt_token (SameSite khi đăng xuất)
  • Mật khẩu: BCrypt qua PasswordEncoder
Email / SMTP

Cấu hình trong application.properties (Gmail SMTP). Quên mật khẩu & gửi mã qua ResetPasswordService. Reset do Admin không bắt buộc email — dùng API reset trả mật khẩu tạm về client (chỉ dùng nội bộ an toàn).

CSDL

MySQL schema hrm_system, Hibernate ddl-auto theo môi trường dev. Bảng taikhoan.ngay_tao dùng cho thống kê tài khoản tạo trong tháng.

Thông báo chung (broadcast)

Gửi tới /topic/notifications + lưu bảng thong_bao. Mọi user (ADMIN/HR/EMPLOYEE) đang mở dashboard đều nhận realtime.
